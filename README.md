# KSS 2026 – Knowledge Sharing System

This repository hosts the KSS 2026 internal portal.

The portal is designed to:
- Display weekly KSS schedules
- Highlight topics planned by management
- Show ownership and status clearly
- Provide a single reference point for the team

## Technology Used
- HTML – for structure and content
- CSS – for styling and layout

## Hosting
This website is hosted using **GitHub Pages** as a static site.
No backend or server-side code is used.

## How to Update
1. Update `index.html` locally
2. Commit changes to the `main` branch
3. GitHub Pages automatically reflects the updates

## Purpose
This portal is intended for internal reference to improve
visibility, consistency, and engagement in weekly KSS sessions.
